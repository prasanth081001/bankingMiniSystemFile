# 🏦 Banking Mini Core System

## 📌 Project Overview
This project is a simple Banking System built using Spring Boot and MySQL.  
It allows users to create accounts, perform transactions, and view account statements.

---

## 🚀 Features

✅ Create Account  
✅ Deposit Money  
✅ Withdraw Money  
✅ Transfer Funds  
✅ Transaction History  
✅ Statement API (DTO Based)  
✅ Running Balance Calculation  
✅ Credit / Debit Identification  

---

## 🛠️ Tech Stack

- Java 17+
- Spring Boot
- Spring Data JPA
- MySQL
- IntelliJ IDEA
- Postman (API Testing)
